# Blinkit Sales Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive Power BI dashboard built using the Blinkit sales dataset. The dashboard provides insights into sales performance, customer purchasing behavior, outlet performance, and product trends through interactive visualizations and KPIs.

---

## 📊 Dashboard Features

- Total Sales
- Average Sales
- Number of Items Sold
- Average Rating
- Sales by Fat Content
- Sales by Item Type
- Sales by Outlet Size
- Sales by Outlet Location
- Sales by Outlet Type
- Outlet Establishment Trend
- Interactive Filters (Outlet Location, Outlet Size, Item Type)

---

## 🛠 Tools Used

- Power BI
- Power Query
- DAX
- Microsoft Excel

---

## 📈 Key Insights

- Tier 3 outlets generated the highest overall sales.
- Regular fat products contributed more revenue than low-fat products.
- Medium-sized outlets achieved the highest sales.
- Supermarket Type 1 outlets recorded the highest revenue among all outlet types.
- Fruits & Vegetables and Snack Foods were among the top-performing product categories.
